# 🍎 Apple Premium Store - Frontend

Template reutilizable para crear tiendas informativas de productos Apple (o cualquier tipo de negocio). Sitio 100% estático, moderno y visualmente atractivo.

## ✨ Características

- 🎨 **Diseño moderno** inspirado en apple.com
- 📱 **Totalmente responsive**
- ⚡ **Rendimiento óptimo** con Next.js 14
- 🎭 **Animaciones suaves** con Framer Motion
- 🎯 **Fácil personalización** - Solo edita archivos JSON
- 🚀 **Deploy gratuito** en Vercel

## 🚀 Inicio Rápido

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)
